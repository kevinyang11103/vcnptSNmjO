# 大学生就业平台微信小程序+SSM

## 前言

随着时代的发展，大学生就业问题日益凸显，为了更好地服务大学生群体，我们团队开发了这款基于微信小程序的就业平台。本项目利用SSM框架和Uniapp技术，致力于为大学生提供便捷、高效的就业信息查询与匹配服务。

## 内容介绍

本项目主要包括以下几个功能模块：个人信息管理、职位搜索、职位推荐、在线投递、消息通知等。用户可以通过微信小程序轻松实现求职需求，同时企业也可以发布招聘信息，实现双向互动。此外，我们还提供了丰富的数据统计和分析功能，帮助用户更好地了解就业市场动态。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
